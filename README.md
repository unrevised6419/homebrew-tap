# Unrevised6419 Tap

## How do I install these formulae?

`brew install unrevised6419/tap/<formula>`

Or `brew tap unrevised6419/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "unrevised6419/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
